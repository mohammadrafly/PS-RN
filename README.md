## Getting Started

Ikuti perintah berikut agar Aplikasi RN Expo bisa di run.

### Built With

* [![React][React.js]][React-url]

### Prerequisites

Penting! install bahan dibawah ini:
* npm 16^
* expo-go App (donglot di playstore)

### Installation

* Install Dependencies
   ```sh
   npm install
   ```
* Run Expo All Devices
   ```sh
   npm expo start
   ```
* Run Expo On Android
   ```sh
   npm run android
   ```
